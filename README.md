# NLP Assignments

## 프로젝트 소개

이 저장소는 자연언어처리 과목에서 수행한 과제와 실습 내용을 정리한 저장소입니다.

자연어 데이터 전처리, 텍스트 분석, 토큰화, 임베딩, 분류 모델, 딥러닝 기반 자연어처리 실습 등을 과제별로 정리하는 것을 목표로 합니다.

## 주요 내용

- 자연어 데이터 전처리
- 토큰화 및 정제
- 단어 빈도 분석
- 텍스트 벡터화
- 머신러닝 기반 텍스트 분류
- 딥러닝 기반 자연어처리 모델 실습
- 과제별 코드 및 결과 정리
